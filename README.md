# Menu Hambúrguer

## Menu Responsivo Versão Desktop: 

### Os links são exibidos horizontalmente, com um espaçamento adequado entre eles, ao passar o mouse, os links mudam de cor.

![Telabig](https://github.com/user-attachments/assets/d5f876f6-b378-4c27-983a-5d6fa31db68a)

## Versão Mobile:

### Um ícone de hambúrguer normalmente aparece no canto superior direito.

![menu](https://github.com/user-attachments/assets/e2e62612-edcc-4e1c-a5bc-ebe1d464b192)

### Ao clicar no ícone, um menu vertical se expande, ocupando a tela. Os links podem ser dispostos em uma lista com um fundo que contrasta com o resto do site. Uma opção de fechar o menu, geralmente um “X”, é exibida no canto superior.

![menuopem](https://github.com/user-attachments/assets/9d078489-7ae4-47ff-8398-d7a39e5990e5)

## Como fez

### Apenas alterando o estado da minha variável se for true adicione a classe que muda o menu hambúrguer e a que faz o menu ir a lateral se for false remove a classe e volta a ser como antes
